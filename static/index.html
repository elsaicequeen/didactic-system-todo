Views Inbox = unsorted tasks. Today = due today + overdue. Upcoming = future due dates. All = everything.
Add a task Click the + bar or press Q. Set priority, project, and due date inline before saving. Press Enter to submit fast.
Priority P1 urgent · P2 high · P3 medium · P4 normal. Tasks sort by priority automatically.
Projects Click Add project below, pick a color. Tasks added from inside a project view inherit that project. Reassign anytime via the edit modal.
Due dates Red = overdue. Orange = due today. Grey = future. Overdue tasks appear in Today view automatically.
Edit a task Click the task title or the pencil icon on hover. The edit modal lets you change everything including description and project.
Project colors are saved in the database and sync across all devices.
